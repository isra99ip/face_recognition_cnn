FaceID en el Aula

CNN + TFLite + Android

Problema

Identificación en tiempo real de integrantes del equipo con latencia < 500 ms.

Dataset

5 clases × ≥100 imágenes por clase. Variación de pose, iluminación y expresión.

Arquitectura

MobileNetV2 → Fine‑tuning → TFLite (float16/INT8) → CameraX.
